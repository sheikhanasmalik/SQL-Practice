# SQL Practice

This repository contains SQL practice scripts designed to strengthen database management and data analysis skills. It includes examples of:

* Database and Table Creation (DDL)
* Data Insertion, Update, and Deletion (DML)
* Filtering and Sorting Data
* Aggregate Functions
* Joins
* Subqueries
* Views
* Window Functions

## Technologies Used

* SQL
* PostgreSQL
